# My Home Library

A comprehensive Flutter mobile application for managing your personal book collection and wishlist. Organize, track, and discover books with an intuitive interface and powerful features.

## Features

### Library & Wishlist Management
- **Dual Collection System**: Maintain separate libraries for books you own and wishlist items
- **Series Grouping**: Automatically group books by series for easy navigation
- **Author Grouping**: View your collection organized by author
- **Flexible Grouping Options**: Toggle between no grouping, series grouping, or author grouping

### Search & Discovery
- **Google Books Integration**: Search for books using the Google Books API
- **Barcode Scanner**: Quickly add books by scanning ISBN barcodes with your camera
- **Search History**: View and reuse your last 5 searches for quick access

### Book Management
- **Detailed Book Information**: View title, author, publisher, publication date, genre, ISBN, page count, and descriptions
- **Personal Notes**: Add custom notes to any book in your collection
- **Read Status Tracking**: Mark books as read or unread
- **Star Ratings**: Rate books from 1 to 5 stars
- **Series Assignment**: Organize books into series with autocomplete suggestions

### User Experience
- **Dark Mode**: Toggle between light and dark themes with persistent preferences
- **Grid & List Views**: Switch between grid and list layouts for browsing
- **Advanced Filtering**: Filter by read status, rating, genre, or search query
- **Multiple Sort Options**: Sort by title, author, rating, date added, or genre (ascending/descending)
- **Wishlist Priority**: Set priorities for wishlist items and reorder them

### Data Management
- **Local Storage**: All data stored locally using SQLite
- **Persistent Settings**: Dark mode and search history preferences are saved
- **Data Removal**: Clear all data option with confirmation dialog
